📋 Lista de Tarefas

Uma aplicação simples para gerenciar suas tarefas diárias via terminal

📖 Sobre o projeto
Este é um projeto de Lista de Tarefas desenvolvido em Java que permite gerenciar tarefas do dia a dia diretamente pelo terminal, sem necessidade de internet, cadastro ou instalação de dependências externas.
🚀 Funcionalidades

 Adicionar tarefa
 Listar tarefas
 Remover tarefa
 Sair do programa

🛠️ Tecnologias utilizadas

Java SE - Linguagem de programação
VS Code - Editor de código
ArrayList - Estrutura de dados para armazenamento em memória
Scanner - Captura de inputs do usuário

📦 Instalação

Pré-requisitos:

Instale o Java JDK (versão 11 ou superior)


Clone o repositório:

   git clone https://github.com/Evelynmctech/ListaDeTarefas.git

Acesse o diretório do projeto:

   cd ListaDeTarefas

Compile o programa:

   javac ListaDeTarefas.java

Execute o programa:

   java ListaDeTarefas
🎯 Como usar

Abra o terminal
Execute: java ListaDeTarefas
O programa irá mostrar as opções:

Digite 1 para adicionar uma tarefa
Digite 2 para listar todas as tarefas
Digite 3 para remover uma tarefa
Digite 0 para sair
