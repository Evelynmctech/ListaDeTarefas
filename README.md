# 📋 Lista de Tarefas

> Uma aplicação simples para gerenciar suas tarefas diárias

## 📖 Sobre o projeto

Este é um projeto de **Lista de Tarefas** (To-Do List) que permite:
- ✅ Adicionar novas tarefas
- ✅ Marcar tarefas como concluídas
- ✅ Excluir tarefas
- ✅ Visualizar todas as tarefas

## 🚀 Funcionalidades

- [x] Criar tarefa
- [x] Listar tarefas
- [x] Marcar tarefa como concluída
- [x] Excluir tarefa

## 🛠️ Tecnologias utilizadas

- **[Java](https://www.java.com/)** - Linguagem de programação
- **[VS Code](https://code.visualstudio.com/)** - Editor de código

## 📦 Instalação

1. **Pré-requisitos:**
   - Instale o [Java JDK](https://www.oracle.com/java/technologies/downloads/) (versão 11 ou superior)

2. **Clone o repositório:**
```bash
git clone https://github.com/Evelynmctech/ListaDeTarefas.git
```

3. **Acesse o diretório do projeto:**
```bash
cd ListaDeTarefas
```

4. **Compile o programa:**
```bash
javac main.java
```
(Substitua `main.java` pelo nome do seu arquivo principal)

5. **Execute o programa:**
```bash
java main
```
(Substitua `main` pelo nome da sua classe principal, sem `.java`)

## 🎯 Como usar

1. Abra o terminal/VS Code
2. Execute: `java main`
3. O programa irá mostrar as opções:
   - Digite `1` para adicionar tarefa
   - Digite `2` para listar tarefas
   - Digite `3` para concluir tarefa
   - Digite `4` para excluir tarefa
   - Digite `0` para sair
